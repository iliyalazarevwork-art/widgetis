Сумка на руку Crivit рожева IAN297343 pink

Опис товару:

  • Сумка має компактні розміри та призначена для транспортування телефона, плеєра, під час бігу або пішій прогулянки. 
  • Регульований ремінець на липучці.
  • Для багатьох поширених смартфонів і MP3-плеєрів.
  • З великим оглядовим вікном — можливе керування сенсорним екраном через прозору плівку.
  • Просте регулювання ширини за допомогою застібки-липучки (від 22 до 34 см).
  • Для пристроїв розміром приблизно до Д 16 x Ш 8 см.
  • Світловідбивні елементи.
  • Тильна сторона також рожевого кольору.

Характеристики:

  • Матеріал: поліестер, еластан. 
  • Колір: рожевий.
  • Зовнішні розміри ШхВхГ (см): 46 x 17 см.
  • Для пристроїв розміром приблизно до Д 16 x Ш 8 см.
  • Вес(кг): 0.042.
  • Виробник: Crivit.
